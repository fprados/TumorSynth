# TumorSynth
mri_TumorSynth - To segment the healthy brain tissue and tumor in MR scans with tumor with support for multi-sequence MR inputs (T1, T1CE, T2, FLAIR). [See TumorSynth website](https://surfer.nmr.mgh.harvard.edu/fswiki/TumorSynth)
